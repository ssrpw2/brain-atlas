Cerebellar hemispheres

Quick Start

  1. Download and install Blender (free, version 4.0+)
  2. Download this repo as a ZIP (green Code button on GitHub)
  3. Open brain_atlas.blend in Blender
  4. Orbit with middle-mouse drag, zoom with scroll, toggle lobes with eye icons in the Outliner
